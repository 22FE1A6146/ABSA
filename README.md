
# Aspect-Based Sentiment Analysis (ABSA)

This project implements an Aspect-Based Sentiment Analysis (ABSA) system using a hybrid **RoBERTa-large + LLaMA** model to perform fine-grained sentiment classification. The model identifies specific aspects within text and predicts sentiment polarity (positive, negative, neutral) at the aspect level.

The system was trained and evaluated on multiple benchmark datasets to improve contextual understanding and classification accuracy. It leverages transformer-based embeddings and advanced language modeling techniques for enhanced performance.

## Tech Stack

* Python
* PyTorch / Transformers
* RoBERTa-large
* LLaMA
* NLP & Deep Learning

## Features

* Aspect-level sentiment detection
* Multi-dataset training
* Transformer-based contextual modeling
